# 2026-06-21 阶段1：源码全景与项目结构

## 今天学了什么

- Vue 3 源码的 Monorepo 结构（pnpm workspace）
- 三层架构：Reactivity → Runtime → Compiler
- 平台无关（core）vs 平台相关（dom）的分层设计
- `vue` 入口包的组装方式

## 关键代码片段

### packages/vue/src/index.ts - 入口包的组装

```typescript
// 完整版 = 运行时 + 编译器
import { compile } from '@vue/compiler-dom'
import { registerRuntimeCompiler } from '@vue/runtime-dom'

// 把编译器注册到运行时
registerRuntimeCompiler(compileToFunction)

// 重新导出所有 runtime-dom API
export * from '@vue/runtime-dom'
```

### 核心包依赖关系

```
vue (入口)
├── @vue/compiler-dom → @vue/compiler-core
├── @vue/runtime-dom → @vue/runtime-core → @vue/reactivity
├── @vue/compiler-sfc
└── @vue/shared (被所有包依赖)
```

## 面试话术

> "Vue 3 采用分层架构，核心分为三层：Reactivity（响应式）、Runtime（运行时）、Compiler（编译器）。每层都有 core（平台无关）和 dom（浏览器专用）的分离，这样可以轻松适配不同平台，如 SSR、小程序等。"
